# book2mk.py脚本使用说明

## 功能介绍
book2mk.py是一个用于从豆瓣读书网站爬取书籍信息并生成Markdown文件的工具。它可以自动提取书籍的核心信息，包括：

- 书名
- 作者
- 出版年份
- 出版社
- ISBN号

## 使用方法
### 基本命令格式
```
uv run book2mk.py <书名>
```
### 示例
```
uv run book2mk.py 三体
```
### 输出结果
执行上述命令后，工具会：

1. 搜索豆瓣读书中的"三体"相关书籍
2. 提取第一本匹配书籍的详细信息
3. 在 output 目录下生成 三体.md 文件
生成的Markdown文件格式如下：

```
# 三体

## 书籍基础信息
| 项目 | 内容 |
|------|------|
| 书名 | 三体 |
| 作者 | 刘慈欣 |
| 出版年份 | 2008 |
| 出版社 | 重庆出版社 |
| ISBN | 9787536692930 |

## 说明
本文件由book2MK工具自动生成，数据来源：豆瓣读书
```

## 技术特点
1. 动态内容处理 ：豆瓣读书使用JavaScript动态加载数据，脚本通过提取 window.__DATA__ 变量中的JSON数据来获取搜索结果
2. 多方法信息提取 ：对于关键信息（如出版年份、出版社、ISBN），脚本实现了多种提取方式，提高了数据获取的成功率
3. 错误处理 ：脚本包含了完善的错误处理机制，能够应对网络请求失败、数据解析错误等情况
4. 调试信息 ：脚本会输出详细的调试信息，方便排查问题

## 注意事项
1. 请确保已安装依赖： uv install
2. 网络环境需要能够访问豆瓣网站
3. 由于网络或豆瓣网站结构变化，部分信息可能无法获取
如果遇到问题，可以查看脚本输出的调试信息，帮助定位问题所在。
